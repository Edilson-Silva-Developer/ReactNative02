URL

Cada arquivo que carrefa no navegador é nomeado por uma URL; A URL (Uniform REsouce Locator) pode ser dividida em três partes:
protocolo,servidor e arquivo;
http://www.horadecodar.com.br/index.html
http = protocolo;
www.horadecodar.com.br = servidor ( DNS para um IP);
index.html = arquivo;
