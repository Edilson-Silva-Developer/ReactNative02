Podemos definir valores com VAr e CONST

var nome='Edilson';
console.log(nome)


const sobrenome = 'Antonio';//CONST não pode ser alterado igual ao LET
console.log(sobrenome);



VAR é uma fomra mais antiga, você deve optar por LET ou CONST
CONST vem de constante, ou seja o seu valor não pode ser alterado.
