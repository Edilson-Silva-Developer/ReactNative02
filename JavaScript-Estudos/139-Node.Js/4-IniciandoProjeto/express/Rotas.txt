O que são Rotas

NOs sites e nas APIs temos o que chamamos de rotas;
Que é basicamente a URL que queremos acessar;
Porém quando tratamos de uma API chamamos as URLs de rotas;
NO Express vamos criar as nossas rotas para serem acessadas via HTTP e responderem
baseadas numa lógica que inserimos;
E casa rota vai receber um método HTTO, que deve coincidir com o proposto
pelo REST, conforme vimos anteriomente.
