O que é Node ?

Não é uma linguagem de programação;

Uma ferramenta construida sobre o motor JavaScript do google
chome para criar aplicações rápidas e escaláveis;

O problema: cada conexão de um cliente aloca um espaço na memoria;

Com Node: casa conexão é uma evento executado na engine no Node;

Ou seja, o Node suporta muito mais conexões do que PHP e Java em uma 
máquina igual, por exemplo;
